❗ USAGE
Setup: Place the bot.py and .env files into the same folder. Open the .env file and enter your bot token.

Permissions: Configure the roles authorized to post links within the bot.py file.

Prerequisites: Ensure that FFmpeg is installed on your system.


# discord-bot
🇹🇷🇹🇷🇹🇷

🎫 Ticket Sistemi
/ticketkur [kanal] — Belirtilen kanala ticket paneli gönderir. Panel üzerinden kullanıcılar 6 farklı kategoride ticket açabilir.
Kategoriler:

🛒 Satın Alma / Purchase
🙋 Destek Yardım / Support
🔧 Teknik Destek
🎬 Media Başvuru / Media Application
📋 Kadro Başvuru / Staff Application
💬 Diğer / Other
❗Her kullanıcı aynı kategoride yalnızca bir ticket açabilir. Ticket içinde "Ticket'ı Kapat" butonu ile yetkili kapatar ve kanal otomatik  silinir.

📢 Güncelleme Duyurusu
/guncelleme [metin] — @everyone etiketleyerek duyuru kanalına embed duyuru gönderir.
/guncelleme_kur [kanal] — Duyuruların hangi kanala gönderileceğini ayarlar. (Yönetici)

🎉 Çekiliş
/giveaway [süre] [kazanan] [ödül] — Belirtilen süre (saniye), kazanan sayısı ve ödülle çekiliş başlatır. Kullanıcılar 🎉 tepkisiyle katılır, süre bitince kazananlar rastgele seçilir. (Yönetici)

🎵 Müzik
/song [link] — YouTube veya YouTube Music linkinden müzik çalar. Playlist linki verirsen tüm şarkılar kuyruğa eklenir.
/skip — Çalan parçayı atlayıp sıradakine geçer.
/pause — Müziği duraklatır.
/resume — Duraklatılan müziği devam ettirir.
/stop — Müziği durdurur, kuyruğu temizler ve ses kanalından çıkar.
/queue — Kuyruktaki şarkıları listeler (max 15 gösterir).

🛡️ Otomatik Koruma
Link koruması — İzinsiz kullanıcı link paylaşırsa mesaj otomatik silinir ve 60 saniye susturulur. Yetkili roller bu kısıtlamadan muaftır.

#discord-bot
ENGLİSH

🎫 Ticket System
/ticketset [channel] — Sends the ticket panel to the specified channel. Users can open tickets in 6 different categories via the panel.
Categories:

🛒 Purchase!

🙋 Support

🔧 Technical Support

🎬 Media Application

📋 Staff Application

💬 Other

Note: Each user can only open one active ticket per category. The "Close Ticket" button allows authorized staff to close the ticket, which automatically deletes the channel.

📢 Update Announcements
/update [text] — Sends an embed announcement to the update channel, tagging @everyone.
/update_setup [channel] — Configures the destination channel for announcements. (Administrator only)

🎉 Giveaway
/giveaway [duration] [winners] [prize] — Starts a giveaway with a specified duration (in seconds), number of winners, and prize. Users participate by reacting with 🎉. Winners are selected randomly once the timer expires. (Administrator only)

🎵 Music
/song [link] — Plays music from YouTube or YouTube Music. If a playlist link is provided, all songs are added to the queue.
/skip — Skips the current track and moves to the next one.
/pause — Pauses the music.
/resume — Resumes the paused music.
/stop — Stops the music, clears the queue, and disconnects the bot from the voice channel.
/queue — Displays the current song queue (shows up to 15 tracks).

🛡️ Auto-Protection
Link Protection — Automatically deletes links posted by unauthorized users and mutes them for 60 seconds. Staff roles are exempt from this restriction.

Note: Additionally, when bot.py is executed, the command prompt (CMD) provides real-time logs, displaying every action performed by the bot in detail.
